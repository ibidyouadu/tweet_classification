This contains all data, code, results, etc. for the tweet classification project.

For raw tweets, keyword scoring, and labeling results, see the data folder. Be careful before using many of the results for analysis however. They may need to be redone in some cases. Consult the readme's within

The documentation is written in LaTeX. The pdf and source code can be found in the documentation folder along with the used media.

For random forest and BERT model notebooks and their results, see the modeling folder. Unlike the previous folder, all the stuff there is up to date and has been carefully maintained.

power_outages is not related to classification exactly, but is the next step in the project: using the refined, labeled, power-related twitter data to analyze outage data. It contains only data from Florida during Irma
